
С помощью Postman создать коллекцию для тестирования следующего сценария для сервиса https://dev.cakeiteasy.se/: (https://dev.cakeiteasy.se/)
 • Регистрация пользователя
 • Выбор города для заказа
 • Добавление товара в корзину
 • Получение списка товаров в корзине
 • Выход из учетной записи
 • Получение данных корзины
 • Авторизация пользователя
 • Оформление заказа
 • Оплата заказа
В данном задании я выполняла интеграционное тестирование между модулями внутри системы, а также между сервом оплаты, проводила оплату заказов чере
з сервис Bambora. Кроме этого, я создавала окружение, объявляла переменные и сохраняла их значение к окружение, писала тест-скрипты для получения рандомных тестовых данных и записи их в перменные окружения, вытаскивала ID пользователя, ID оплаты из тела ответа от сервера и переиспользовала их в последующих запросах.
